## 🎯 Sobre o Projeto

Este site foi desenhado sob medida para ser leve, responsivo e altamente visual. Focado na **cadeia produtiva do leite**, ele reúne conhecimento de manejo, tecnologia de ordenha e nutrição de precisão em uma interface limpa e moderna.

> 💡 **Destaque Técnico:** O projeto agora conta com **Dark Mode nativo** programado em JavaScript puro e um layout em **Grid** totalmente adaptável para telas de smartphones.
