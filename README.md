# 🥛 Blog do Kauã - Bovinocultura de Leite

Bem-vindo ao repositório do meu blog pessoal focado em **Bovinocultura de Leite**! Este espaço foi desenvolvido para compartilhar conhecimento prático, dicas de manejo e novidades do setor produtivo leiteiro.

## 🚀 Tecnologias Utilizadas

O blog foi construído utilizando tecnologias web fundamentais, mantendo o código limpo, aprimorado e bem organizado:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização moderna, responsiva e com foco na experiência visual.
* **JavaScript:** Pequenas interações para dinamizar a leitura.

## 📁 Estrutura de Arquivos

O projeto está dividido separadamente para facilitar modificações futuras:

```text
├── index.html   # Estrutura e textos do blog
├── style.css    # Cores, fontes e design visual
